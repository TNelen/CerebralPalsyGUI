For more information about the GUI, you can press the help-button in the top right corner of its screen. A window will open where you can find an explanation of every function.
The GUI is named GUI.mlapp and must always be in the same directory as Help.mlapp. Otherwise, the help-button would not work.

The fft can be found in calculate.m. Two variables have been included: X.mat and Y.mat which are healthy data and cerebral palsy data, respectively.
